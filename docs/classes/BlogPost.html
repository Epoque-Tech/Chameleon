__construct(array  $asso_array)

The BlogPost Constructor

Takes an associative array with each key being a BlogPost property (i.e. title, author, etc) with it's corresponding value. For each key that that is a class property the constructor sets that key's value to the object's property value.

Parameters

array $asso_array

An associative array with key/value pairs representing attributes of a blog post.
